## 🛠️ Tech Stack

- **Framework:** React.js (Vite)
- **Styling:** Tailwind CSS
- **3D Graphics:** Three.js
- **Routing:** React Router
- **UI Enhancements:**
  - Framer Motion
  - AOS (Animate On Scroll)
  - React Awesome Reveal
  - React Icons
  - React Fast Marquee
  - React Loading Skeleton
  - React Typewriter
- **Email Integration:** Email.js
- **Deployment:** Vercel
- **Version Control:** Git + GitHub
